# Assets

This directory contains supporting material for lessons: diagrams, screenshots, and small packet captures that help explain what a lab observed.

日本語: このディレクトリには、Labの理解を助ける図、スクリーンショット、小さなpcapなどを入れます。

## Guidelines

- Keep packet captures small and focused on the lesson.
- Use one subdirectory per lab, such as `assets/bgp-01/`.
- Review captures and screenshots before sharing them. They should not include secrets, local hostnames, credentials, or unrelated traffic.

日本語:

- pcapは小さく、Labで見る内容に絞る。
- `assets/bgp-01/` のようにLabごとに分ける。
- 共有前に、秘密情報、ローカルホスト名、認証情報、無関係な通信が入っていないか確認する。
