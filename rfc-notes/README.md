# RFC Notes

RFC notes are reading guides for the labs. They do not replace the RFC. They help you decide which sections to read first and how those sections connect to observable behavior in the lab.

日本語: RFC Notes は、Labのための reading guide です。RFC本文の代わりではありません。どの章を先に読み、Labで観察する出力とどう対応づけるかを示します。

## Available Guides

- [RFC 4271 Reading Guide: BGP-4](bgp-rfc4271.md)

## How to Use Them

- Open the RFC alongside the note.
- Read the listed sections before running the lab.
- Come back after the lab and map command output or packets back to RFC terms.

日本語:

- RFC本文と note を並べて読む。
- Labを動かす前に、指定された章を読む。
- 実験後に、コマンド出力やpcapをRFCの用語へ対応づける。

Quoted text should stay short. Most explanations should be written in the reader's own words.

日本語: 引用は短くし、説明はできるだけ自分の言葉で整理します。
